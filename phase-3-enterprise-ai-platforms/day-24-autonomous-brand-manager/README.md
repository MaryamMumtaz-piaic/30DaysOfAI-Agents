## 📖 Overview

A 24/7 autonomous brand management platform. Monitors all social channels, auto-responds to comments and DMs with brand-voice enforcement, creates weekly content calendars, A/B tests post variations, tracks brand sentiment trends, detects influencer mentions, and generates weekly brand health reports with competitor benchmarking.

## ✨ Key Features

- Multi-platform social integration (Twitter, Instagram, LinkedIn)
- Brand-voice-trained response system with tone enforcement
- AI content calendar generation with optimal posting times
- A/B post testing with automated performance tracking
- Weekly brand health PDF report with competitor analysis

## 🛠️ Tech Stack

Python · CrewAI · OpenAI SDK · Social APIs · FastAPI · HTML · Tailwind CSS · JavaScript · PostgreSQL · Celery + Redis

## ⚙️ Setup

```bash
cd phase-3-enterprise-ai-platforms/day-24-autonomous-brand-manager
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp ../../.env.example .env       # add your API keys
docker compose up --build
```
