## 📖 Overview

The ultimate capstone: a unified AI Command Center that orchestrates ALL 29 agents built throughout the challenge. Deploy any agent, monitor all running workflows in real-time with live logs, measure performance metrics, chain agents together on-the-fly, manage resource usage and API costs, and control everything from one premium interface. The Jarvis of AI systems.

## ✨ Key Features

- Single dashboard controlling and monitoring all 29 prior agents
- Real-time agent health monitoring with live execution logs
- On-the-fly agent chaining and dynamic pipeline creation
- Resource usage and API cost tracking per agent/user
- Master REST API gateway for external programmatic access
- Role-based access control (RBAC) with team management

## 🛠️ Tech Stack

Python · LangGraph · CrewAI · FastAPI · HTML · Tailwind CSS · JavaScript · WebSockets · PostgreSQL · Redis · Docker Compose · Nginx

## ⚙️ Setup

```bash
cd phase-3-enterprise-ai-platforms/day-30-ai-command-control-center
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp ../../.env.example .env       # add your API keys
docker compose up --build
```
